Adding Kadence Voice to an Existing Image

This example shows how to attach Kadence voice functionality to an existing image on your LearnWorlds page. When users click the image, the voice assistant will activate.

Live Example:

Click the Kadence logo below to start the voice assistant:

The red pulse overlay will appear when active.

How to Implement in LearnWorlds

To add this functionality to an image in your LearnWorlds site, follow these steps:

Step 1: Add the Script

Add this code to your LearnWorlds page using the Custom Code element:

<!-- Load the required scripts -->
<script src="https://kadence-multimodal.vercel.app/embed/kadence-inline.js"></script>

Step 2: Initialize on Your Image

Add this code after your image (replace yourImageId with the ID of your image):

<script>
// Initialize Kadence on your image 
// You can use any selector (ID, class, etc.)
document.addEventListener('DOMContentLoaded', function() {
  window.initKadenceOnElement('#yourImageId', {
    statusPosition: 'overlay', // 'overlay', 'after', 'before'
    statusStyle: 'pulse'       // 'pulse', 'dot', 'indicator'
  });
});
</script>

Step 3: Add an ID to Your Image

Make sure your image has an ID attribute that matches the selector used above:

<img id="yourImageId" src="your-image-url.jpg" alt="Click to talk to Kadence">

Important:

  • You can use this approach with any element, not just images (buttons, divs, etc.)
  • The overlay option works best with images that have a good amount of space
  • You can customize the visual indicator style using the options
  • The microphone permissions prompt will appear when the user first clicks the image

Customization Options

The initKadenceOnElement function accepts these options:

window.initKadenceOnElement('#elementSelector', {
  // Where to place the status indicator
  statusPosition: 'overlay',  // 'overlay', 'after', 'before'
  
  // Style of the status indicator
  statusStyle: 'pulse',       // 'pulse', 'dot', 'indicator'
  
  // Start automatically without waiting for click
  autoStart: false            // true, false
});
