Kadence Voice Embedding Examples

This page demonstrates how to embed the Kadence voice assistant in your LearnWorlds course pages. There are two methods available for embedding:

Method 1: Embed a Button

Add a button that when clicked will activate the voice assistant. This is the recommended approach for most LearnWorlds courses.

Live Example:

How to implement:

Add this code to your LearnWorlds page using the Custom Code element:

<!-- 1. Add a container where the button should appear -->
<div class="kadence-button-container" data-text="Talk to Kadence AI"></div>

<!-- 2. Load the Kadence button script -->
<script src="https://kadence-multimodal.vercel.app/embed/kadence-button.js"></script>

Method 2: Direct Widget Embedding

Directly embed the voice assistant widget without requiring a button click first. The widget appears as a floating microphone button in the corner of the page.

Live Example:

Click the microphone button in the bottom-right corner to activate:

How to implement:

Add this code to your LearnWorlds page using the Custom Code element:

<!-- Simply load the Kadence voice script -->
<script src="https://kadence-multimodal.vercel.app/embed/kadence-voice.js"></script>

Important Notes:

  • Microphone Access: The browser will ask for microphone permission when the user first interacts with Kadence.
  • HTTPS Required: The voice assistant will only work on secure (HTTPS) connections.
  • Browser Compatibility: Works best in Chrome, Edge, Firefox, and Safari on desktop. Mobile support may vary.
  • Username: The voice assistant will try to detect the user's name from LearnWorlds, falling back to "student" if not found.

Customization

For developers who want more control, the button can be programmatically created:

// After loading kadence-button.js:
window.KadenceButton.create({
  text: 'Ask Your AI Tutor',      // Custom button text
  target: '#my-container',        // CSS selector or DOM element 
  position: 'beforeend'           // DOM insertion position
});
